Donanım ve API

ESP32 Mega ile seri (9600) konuşur; bu bilgisayarda çalışan Spring Boot API'sine HTTP ile bağlanır. Cihaz ID aşağıdaki tablodaki sayı ile aynı olmalıdır (ESP kodundaki DEVICE_DB_ID). Kart UID'si veritabanındaki Kart numarası ile eşleşmelidir (küçük harf, boşluksuz hex örn. 04e9b3a2). Tuş takımında okul/sicil numarası girip # ile onaylayın. Parmak izi için kullanıcıya sensördeki slot numarasını kaydedin.
